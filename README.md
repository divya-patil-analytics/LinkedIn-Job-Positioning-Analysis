# LinkedIn Job Postings Analysis

## Project Overview
This project analyzes LinkedIn job postings to identify current hiring trends in the data analytics domain. The analysis focuses on in-demand job roles, experience requirements, salary patterns, skills demand
and hiring locations.

## Tools Used
- Excel – Data cleaning and preprocessing  
- SQL – Data analysis and querying  
- Tableau – Interactive dashboard and visual storytelling  

## Dataset
The dataset contains ~1000 LinkedIn job postings with details such as job role, company, location, experience level, salary range, and required skills. Missing values were handled during the cleaning process.

## Key Analysis Performed
- Job role-wise demand analysis  
- Experience level distribution  
- Salary range analysis  
- Location-wise hiring trends  
- Skill demand overview  

## Dashboard
An interactive Tableau dashboard was created to visualize:
- Top in-demand job roles  
- Salary and experience trends  
- Jobs by location  
- Top Skills 

## Key Insights
- Data Scientist and Machine Learning Engineer roles are the most in-demand.  
- Mid-level experience is preferred across most job postings.  
- Higher salaries are associated with advanced experience levels.  
- Major hiring is concentrated in metro cities.
